Comandos úteis

Comando Git Descrição Exemplo de uso
git init Inicializa um novo repositório Git.
git clone Cria uma cópia de um repositório existente. git clone https://github.com/fcarvalh6/icc-html-1
git branch Lista, cria ou exclui branches. git branch feature-x
git checkout Alterna entre branches ou commits. git checkout feature-x
git status Exibe o estado do diretório de trabalho e da área de stage.
git diff Mostra as diferenças entre commits, entre um commit e a árvore de trabalho, etc.
git add Adiciona o conteúdo de arquivos à área de stage.
git commit Registra as mudanças no repositório.
git push Atualiza referências remotas junto com os objetos associados.
git pull Busca e integra com outro repositório ou branch local.
git revert Cria um novo commit que desfaz as mudanças feitas por um commit anterior.
git merge Mesclar suas alterações de um branch ao branch principal. git merge feature-x
git stash Guarda mudanças no diretório de trabalho sujo temporariamente.
git rm Remove arquivos do diretório de trabalho e da área de stage.
git config Define ou obtém opções de repositório ou globais.
git reset Redefine o HEAD atual para o estado especificado.
git remote Gerencia o conjunto de repositórios rastreados.
git fetch Faz o download de objetos e referências de outro repositório.
git show Mostra vários tipos de objetos.
git help Exibe informações de ajuda sobre o Git.
git rebase Aplica novamente commits no topo de outro branch.
git blame Mostra a revisão e o autor que modificaram cada linha de um arquivo pela última vez.
git tag Cria, lista, exclui ou verifica um objeto de tag assinado com GPG.
